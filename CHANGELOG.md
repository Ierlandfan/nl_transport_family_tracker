# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-12-08

### Added
- Initial release
- UI-based configuration flow with station search
- Real-time departure tracking from OVAPI.nl
- Automatic return trip creation
- Multi-language support (EN, NL)
- Delay detection and notifications
- Multiple departure tracking (configurable 1-10)
- Integration with Life360 for journey monitoring
- Multi-modal transport detection (car/bus/train)
- Unexpected stop detection
- ETA calculations for car journeys
- Smart context-aware notifications

### Features
- Search for stations by name
- Filter by line number and direction
- Track multiple routes simultaneously
- Automatic destination detection
- Journey progress monitoring
- Comprehensive attributes for automations
